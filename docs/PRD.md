# PRD: Webカレンダー（個人練習用）

## 目的
ブラウザだけで使えるシンプルなカレンダーを作り、Cloudflare上でフロント/バックエンドの構築・運用を学ぶ。

## 対象ユーザー
個人（自分のみ）。

## 主要価値
- 予定の作成・閲覧・編集を通じて、UI/状態管理/データ永続化/API設計を学習できる。

## スコープ（MVP）
- 月表示カレンダー
- 予定の作成/編集/削除
- 予定詳細（タイトル/日時/メモ）
- ブラウザのみ対応（PC優先）

## 除外
- 認証
- 共有/招待
- 通知
- 外部連携

## 技術制約
- Cloudflareでフロント/バック構築

## 成功指標
- 予定CRUDが一通り動作する
- Cloudflare上でデプロイできる
